# PRD

## 프로젝트명

yn-customer

---

## 프로젝트 목적

영남수산 고객이 카카오톡 알림톡의 배송조회 버튼을 눌렀을 때 이용하는 모바일 배송조회 웹서비스를 제작한다.

Next.js 기반으로 개발하며 Vercel에 배포할 예정이다.

모바일 사용자를 최우선으로 고려하여 제작한다.

---

## 개발 환경

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Vercel
- Mobile First

---

## 주요 기능

### 배송조회

고객은 운송장번호를 직접 입력하지 않는다.

알림톡의 배송조회 버튼을 누르면 전달받은 안전한 Token을 이용하여 자동으로 배송정보를 조회한다.

---

## 배송조회 흐름

yn-order-manager

↓

알리고 알림톡 발송

↓

고객이 배송조회 버튼 클릭

↓

yn-customer

↓

Token 확인

↓

배송정보 조회

↓

배송정보 표시

---

## 배송조회 화면

화면에는 아래 정보를 표시한다.

- 택배사
- 운송장번호
- 배송상태
- 현재 위치
- 배송 진행 단계

---

## UI 참고

프로젝트 내 **UI 폴더**에 있는

**배송조회 화면 참고.png**

파일을 반드시 먼저 분석한 후 UI를 구현한다.

### 구현 원칙

- 참고 이미지의 레이아웃을 최대한 반영한다.
- 모바일 화면에 맞게 자연스럽게 구현한다.
- 전체적인 디자인, 카드 구성, 여백, 버튼 스타일, 정보 배치 등을 참고한다.

### 추가 요구사항

참고 이미지와 동일한 디자인 방향으로 구현하되,

**상단 좌측에 "배송조회" 버튼을 추가한다.**

배송조회 버튼은 현재 페이지를 나타내는 메뉴 형태로 자연스럽게 배치한다.

---

## 디자인 방향

- 모바일 우선
- 카카오톡 인앱 브라우저 최적화
- 심플하고 깔끔한 디자인
- 영남수산 브랜드에 어울리는 신뢰감 있는 UI
- 큰 글씨와 큰 버튼
- 어르신도 쉽게 사용할 수 있는 화면
